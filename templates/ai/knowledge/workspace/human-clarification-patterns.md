# Human Clarification Patterns

Ask for human input when bootstrap or source evidence cannot safely decide:

- Product semantics or public API behavior
- Data retention, privacy, or release policy
- Conflicting ownership signals
- Destructive migrations
- Missing validation capability for a high-risk change

Prefer one precise question that names the consequence of each option.
